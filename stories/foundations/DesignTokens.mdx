## Dark Mode

O tema escuro é ativado adicionando a classe `.dark` no elemento `<html>`. O toggle do Storybook
(ícone de lua no toolbar) aplica o fundo `#0f1117` para simular o contexto escuro.

```tsx
// Tailwind dark variant configurado via @custom-variant:
@custom-variant dark (&:where(.dark, .dark *));

// Uso nos componentes:
className="bg-card dark:bg-slate-900"
```

> **Nota:** Para ver o dark mode em ação, use o toggle de background no toolbar do Storybook
> e selecione o fundo "dark". Os tokens CSS respondem automaticamente via `:where(.dark, .dark *)`.
