Recomendação

Sistema de duas famílias: Causten para marca (logo, hero de site, capa de decks, peças de marketing) · Geist ou Inter para produto (UI, tabelas, formulários, blog). Após escolher, eu atualizo colors_and_type.css com as duas variáveis (--font-display / --font-sans) e todos os previews herdam.
