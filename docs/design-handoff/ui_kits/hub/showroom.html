Simplafy · Hub Kit

O vocabulário visual do SaaS Simplafy Hub.

Este kit é o "sabor SaaS" da marca Simplafy. Ele não inventa cor, tipo ou brand — tudo isso vive no DS canônico do site. Aqui ficam só os componentes específicos de produto: shell, drawers, agent card, score widget, prompt block, command palette, e os padrões de página.

v0.2 · consome tokens.css v1.1.0 radius alinhado ao canônico ver PATTERNS.md

Como ler: cada componente abaixo mostra (1) o que faz, (2) preview ao vivo no tema atual, (3) quando usar. Alterne o tema no header pra validar nos dois.

05

Page hero

Padrão de abertura de página: eyebrow + Causten + lead + ações.

Hero com eyebrow + mark

.list-hero · .eyebrow · h1 .mark só nas telas-raiz (Agentes, Chat, CRM)

No dark mode, recebe um glow ambiente sutil (radial brand-green a 10% blur 40px) — herança direta do hero do site. Não usar em telas internas (editor, detalhe). Causten 30-44px no Hub (vs. 60-80 no site).

06

Inputs

Tudo o que o usuário toca pra fazer alguma coisa.

Botões

.btn · .btn-primary · .btn-outline · .btn-ghost · .btn-sm

Primary = ação primária da tela (apenas 1). No dark, o primary usa brand-green (neon) com glow sutil. Outline = ação secundária. Ghost = ação terciária / contextual.

Field (label + input)

.field · label · input/select/textarea

Label uppercase letter-spacing pra hierarquia clara. Inputs com radius-sm e foco com ring verde 18%.

Search input

.search-input

Ícone de busca + input. Usado em toolbars de listagem (Agentes, Chat, CRM).

Tab group

.tab-group · .tab · .cnt

Segmented control para filtros de listagem (Todos / Ativos / Inativos). Não usar pra navegação primária (use sidebar pra isso).

Switch

.switch.on

Toggle binário. Usado pra ativar/desativar agente, guardrails, ferramentas.

Chip input

.chip-input · .chip · .chip .x

Lista de tags editável. Enter pra adicionar, X pra remover.
