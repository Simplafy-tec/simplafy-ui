# Hub v2 UI kit

Replica of the Simplafy **Hub v2** dashboard — the core platform product. Uses tokens from `/colors_and_type.css`, Lucide icons from CDN, and logo SVGs from `/assets/logos`.

Screens:
- `index.html` — main dashboard (sidebar + topbar + KPIs + bar chart + agents list + conversations table + "Novo agente" Sheet)

Components captured:
- Sidebar with workspace switcher and nav sections
- Topbar with search, icon buttons, avatar
- MetricCard / KPI grid
- Card with tabs + CSS bar chart
- List pattern (agents)
- Table with inline badges and action menu
- Sheet (side panel) form with Input / Select / Textarea / Switch
